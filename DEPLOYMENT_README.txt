방법 1: AWS Amplify (가장 추천! ⭐⭐⭐⭐⭐)
────────────────────────────────────────────────
• 난이도: ⭐ (초보자도 5분이면 완료!)
• 비용: 무료 ~ $5/월 (프리티어 1년 무료)
• 특징: 
  - GitHub 자동 연동
  - 자동 배포 (git push → 자동 배포!)
  - 무료 SSL 인증서
  - 전 세계 CDN 자동 적용

📖 가이드: AWS_AMPLIFY_SETUP.md

🎯 빠른 시작:
1. https://console.aws.amazon.com/amplify/ 접속
2. "New app" → "Host web app" → "GitHub" 선택
3. 저장소: doogiesdj/NewRIDE, 브랜치: main
4. "Save and deploy" 클릭
5. ☕ 3-5분 기다리면 완료!


방법 2: AWS S3 + CloudFront
────────────────────────────────────────────────
• 난이도: ⭐⭐ (약간의 AWS 지식 필요)
• 비용: $1 ~ $3/월 (매우 저렴!)
• 특징:
  - 빠른 속도 (전 세계 CDN)
  - 무제한 트래픽
  - 완전한 제어 가능

📖 가이드: AWS_HOSTING_GUIDE.md

🎯 빠른 시작 (로컬 PC에서):
1. 저장소 클론: git clone https://github.com/doogiesdj/NewRIDE.git
2. 디렉토리 이동: cd NewRIDE
3. 스크립트 실행: ./deploy_to_s3.sh
4. 버킷 이름 입력 (예: newride-website)
5. ☕ 5분 후 완료!


방법 3: AWS Lightsail
────────────────────────────────────────────────
• 난이도: ⭐⭐⭐ (서버 관리 경험 필요)
• 비용: $3.5/월부터
• 특징:
  - 완전한 서버 제어
  - SSH 접근 가능
  - 추가 서비스 설치 가능

📖 가이드: AWS_HOSTING_GUIDE.md

🔄 업데이트 방법

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Amplify (자동 배포):
  git add .
  git commit -m "업데이트"
  git push origin main
  → 1-2분 후 자동 배포 완료! ✅

S3 (수동 배포):
  git pull origin main
  ./deploy_to_s3.sh
  ./invalidate_cloudfront.sh [DISTRIBUTION_ID]

Lightsail (SSH 업데이트):
  ssh ubuntu@[IP]
  cd /var/www/html
  sudo git pull origin main
